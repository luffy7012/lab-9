TravelApp – Seyahat Rehberi Uygulaması

Bu proje, Mobil Programlama dersi kapsamında verilen React Navigation Seyahat Rehberi Uygulaması ödevi için geliştirilmiştir.  
Uygulamada React Navigation kullanılarak çok ekranlı bir yapı oluşturulmuş, Stack ve Tab Navigatörleri entegre edilmiş ve ekranlar arası veri aktarımı yapılmıştır.
